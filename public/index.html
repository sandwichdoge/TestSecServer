How it works: Each test fetches a server-generated payload through your network. If your security infrastructure (web proxy, next-gen firewall, DLP, antivirus) intercepts and blocks the request, the test passes. A failed test means that threat type could reach your network undetected. No actual malware is used — all payloads are inert test patterns.
